# Vim config
Кастомный переносимый vim конфиг

## Установка

### Загрузка проекта
```
git clone https://github.com/J4CKVVH173/vim_conf.git ~/.vim
ln -s ~/.vim/vimrc.custom ~/.vimrc
```

### Vundle

После установки, произвести запск vim и выполнить в режиме команды выполнить
`:PluginInstall`
